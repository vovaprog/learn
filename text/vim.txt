Delete current line in insert mode:
End (goto end of line), Ctrl + u

Goto line
110 G

indent block of code:
select block, press > or <

==========================================================
From tutorial:

h j k l - move cursor one char or line
i, a - insert, append mode
x - delete char

w - move one word
0 - start of line
$ - end of line
e - end of word

dw - delete word
dd - delete line 

u - undo
U - undo line
Ctrl+r - redo

:wq  - save changes


allow backspace remove previously entered text:
:set backspace=indent,eol,start
or
:set backspace=2

Ctrl+g - get current line

G - end of file
number G - goto line number
gg - first line

/test  - find text
n - find next
N - find prev

?test - find backward

Ctrl + o - move back
Ctrl + i - move forward

% - go to matching bracket

:! ls   - run ls

:w filename -  save to filename

d - cut
y - copy
p - paste

Ctrl+r, потом 0   - paste in insert mode

ignore case in search:
set ic

Open in tab:
tabedit {file}
tabclose

Ctrl + ww   - jump between windows

:qa - close all tabs

:o - insert empty line after current

Ctrl+] - goto definition
Ctrl+t - go back

add ctags:
:set tags+=/path/to/file

build cscope database:
cscope -R -b

==========================================================
Tags:
generate tags:
ctags -R

add tags file:
:set tags+=/path/to/file

Ctrl+] - goto definition
Ctrl+t - go back

goto tag name:
:ta someName

==========================================================

tabs to spaces:
set expandtab

number of spaces in tab:
set tabstop=4

To change all the existing tab characters:
retab

number of space characters inserted for indentation:
set shiftwidth=4

one line:
set tabstop=4 shiftwidth=4 expandtab

set c++ indent:
set autoindent
set cindent


put settings in .vimrc file in home folder:
set tabstop=4 shiftwidth=4 expandtab
set autoindent
set cindent
set nu
set backspace=indent,eol,start
set nocompatible

h j k l - move cursor one char or line

may be used as 4h, 5l.

0 - move to line start
$ - move to line end

line numbers:
set nu

w - move one word forward
b - move one word backward

may be used as 4w, 5b.

33G - goto line 33

a - append mode

Change:
cw To the end of a word
c2b Back two words
c$ To the end of line
c0 To the beginning of line
